# Installation  (Requires NodeJS 16+)
> * Clone the repo
> * Run `npm install`
> * Create a `.env` file with the following contents:
>
> ```env
> DISCORD_TOKEN=YOUR_TOKEN_HERE
> ```
>
> * Run `node .`
